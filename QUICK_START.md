# 🚀 Быстрый старт - 5 минут

## Шаг 1: Установите Node.js (3 минуты)

1. Откройте https://nodejs.org/
2. Нажмите кнопку "LTS" (большая зелёная кнопка)
3. Запустите скачанный файл .msi
4. Нажимайте "Next" → "Finish"

**Проверка:**
```cmd
node --version
npm --version
```

Должны вывести версии (например: v18.16.0)

---

## Шаг 2: Установите приложение (2 минуты)

Откройте Command Prompt или PowerShell:

```bash
cd c:\Users\Asus-2\Downloads\may_project\дока\cube-docs
npm install
```

Это займёт 2-5 минут. Не прерывайте процесс!

---

## Шаг 3: Запустите (1 минута)

```bash
npm run dev
```

✨ Приложение автоматически откроется в браузере на **http://localhost:3000**

## 📖 Первое использование

### Страница "Каталог" (30 секунд)

1. Вы видите вкладку **"📊 Каталог"**
2. В селекторе выберите сущность:
   - **Факты:** F00002 (Планы), F00006 (Скидки)
   - **Измерения:** D00001 (Календарь), D00400 (Скидки), D00750 (SKU)
3. Посмотрите **показатели** (для фактов) или **атрибуты** (для измерений)
4. Посмотрите таблицу **связей**
5. Попробуйте **поиск** - введите "plan"

### Страница "Требования" (2 минуты)

1. Нажмите вкладку **"✏️ Требования"**
2. Введите название: "Мое первое требование"
3. Нажмите **"+ Добавить меру"**
4. Заполните:
   - Название: test_measure
   - Перевод: Тестовая мера
   - Описание: Для теста
   - Выражение: SUM([test])
5. Нажмите **"⚡ Авто"** для создания связей
6. Нажмите **"📋 DOCX"** - скачается файл
7. Нажмите **"💾 Сохранить"**

**Поздравляем!** Вы освоили приложение! 🎉

## 🎯 Основные возможности

### На странице "Каталог"

| Что можно делать | Как |
|-----------------|-----|
| Просмотреть факт | Выбрать F00002 или F00006 |
| Просмотреть измерение | Выбрать D00001, D00400 или D00750 |
| Найти показатель | Ввести название в поиск |
| Посмотреть связи | Прокрутить вниз до таблицы "Связи" |
| Data Lineage | Перейти на вкладку "Data Lineage" (заглушка) |

### На странице "Требования"

| Что можно делать | Как |
|-----------------|-----|
| Создать требование | Заполнить форму сверху |
| Добавить меру | Нажать "+ Добавить меру" |
| Добавить измерение | Нажать "+ Добавить измерение" |
| Связать автоматически | Нажать "⚡ Авто" |
| Экспортировать | Нажать "📋 DOCX" или "📄 Markdown" |
| Сохранить | Нажать "💾 Сохранить требование" |

## ❓ Решение проблем

### npm не найден
→ Убедитесь что Node.js установлен  
→ Перезагрузите PowerShell/CMD

### Ошибка при npm install
→ Попробуйте: `npm cache clean --force`  
→ Затем: `npm install`

### Порт 3000 занят
→ Vite автоматически выберет другой порт

### Приложение не загружается
→ Проверьте консоль браузера (F12)  
→ Возможно есть ошибка в коде
