# 📊 Cube Documentation Tool

Система управления документацией OLAP куба данных.

## 🚀 Быстрый старт

```bash
# 1. Установите Node.js с https://nodejs.org/
# 2. Установите зависимости
cd cube-docs
npm install

# 3. Запустите приложение
npm run dev
```

Приложение откроется на http://localhost:3000

## 📖 Возможности

### Страница "Каталог"
- Просмотр всех фактов (F) и измерений (D)
- Поиск по показателям и атрибутам
- Просмотр связей между таблицами
- Вкладка Data Lineage (DBT/Git) - в разработке

### Страница "Требования"
- Создание функциональных требований
- Добавление новых мер и измерений
- Управление связями
- Экспорт в DOCX и Markdown
- Сохранение в браузер

## 📚 Документация

- **START_HERE.md** - подробное руководство
- **QUICK_START.md** - установка за 5 минут
- **README_RU.md** - краткий обзор

## 🔧 Основные команды

```bash
npm run dev      # Разработка
npm run build    # Production сборка
npm run preview  # Просмотр production версии
```

## 💾 Где хранятся данные

- Тестовые данные: `src/mockData.js`
- Требования: localStorage браузера
- Экспорт: DOCX/MD файлы

## 🎯 Структура

```
cube-docs/
├── src/
│   ├── components/      # React компоненты
│   ├── types.js         # Типы данных
│   ├── mockData.js      # Данные куба
│   ├── exportService.js # Экспорт DOCX/MD
│   └── storageService.js # localStorage
├── index.html
└── package.json
```
