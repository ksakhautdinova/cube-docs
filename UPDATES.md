## 📊 Новая структура данных

### Типы
```javascript
DimensionEntity {
  id: 'D00001',
  name: 'D00001 Основной календарь',
  description: 'Описание',
  tableName: 'd00001_main_calendar',
  columns: [
    { name, translation, description, dataType }
  ]
}

TableRelation {
  fromTable: 'F00002',
  toTable: 'D00001',
  fromColumn: 'plan_date',
  toColumn: 'calendar_date',
  type: 'many-to-one'
}
```

### Меры с переводом
```javascript
{
  id: 'M001',
  name: 'plan_sale_kg',
  translation: 'План продажи, кг',
  description: 'План продажи в килограммах',
  expression: 'TOTALMTD(SUM([plan_kg]), ...)',
  formatString: '#,0.00'
}
```

## 🎯 Использование

### Просмотр факта
1. Выберите `F00002` или `F00006`
2. Увидите **показатели** с DAX формулами
3. Увидите связи с измерениями

### Просмотр измерения
1. Выберите `D00001`, `D00400` или `D00750`
2. Увидите **атрибуты** (столбцы с типами)
3. Увидите связи с фактами

### Поиск
Введите в поиск:
- Название меры (plan_sale_kg)
- Перевод (План продажи)
- Описание (килограммах)

Фильтрация работает мгновенно!
