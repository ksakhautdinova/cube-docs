# 🎯 Последние изменения - Версия 1.1.1

## Что изменилось

### 1. ✅ Поиск объединён с селектором
**Было:** Два отдельных поля - селектор и поиск  
**Стало:** Один блок - селектор сверху, поиск снизу (объединены визуально)

### 2. ✅ Убрано поле "Тип" 
**Было:** В информации о сущности было поле "Тип: 📊 Факт (F) / 📁 Измерение (D)"  
**Стало:** Убрано, т.к. очевидно из выбора в селекторе

### 3. ✅ Убрана колонка "Тип данных"
**Было:** В таблице атрибутов была колонка "Тип данных" (dateTime, string, int64)  
**Стало:** Только 3 колонки - Столбец, Перевод, Описание

### 4. ✅ Упрощена таблица связей

**Было:**
| Направление | Из таблицы | Поле | → | В таблицу | Поле | Тип связи |
|------------|-----------|------|---|-----------|------|-----------|
| → | F00002 | plan_date | → | D00001 | calendar_date | many-to-one |

**Стало:**
| Из таблицы | В таблицу |
|-----------|-----------|
| F00002_pivot_plans_prim([plan_date]) | d00001_main_calendar([calendar_date]) |

**Изменения:**
- Убрана колонка "Направление" (→/←)
- Убрана колонка "Тип связи" (many-to-one)
- Объединены поля в формат: `table_name([column])`
- Показываются реальные имена таблиц БД, а не ID сущностей
