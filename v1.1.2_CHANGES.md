# 🎯 Изменения версии 1.1.2

## Что изменилось

### 1. ✅ Вернули направление связи
**Где:** Таблица "Связи с другими таблицами"  
**Что добавили:** Колонка "Направление" с стрелками → (исходящие) и ← (входящие)

**Пример:**
| Направление | Из таблицы | В таблицу |
|------------|-----------|-----------|
| → | F00002_pivot_plans_prim([plan_date]) | d00001_main_calendar([calendar_date]) |
| → | F00002_pivot_plans_prim([sku_guid]) | d00750_sku([sku_guid]) |

### 2. ✅ Переработали селектор сущностей
**Было:** Обычный select (только выбор)  
**Стало:** Комбобокс (можно вводить + выбирать)

**Как работает:**
1. Можно **ввести название** - начните печатать, список фильтруется
2. Можно **выбрать из списка** - клик открывает dropdown
3. **Поиск в реальном времени** - список фильтруется по мере ввода
4. **Группировка** - Факты и Измерения отдельно
5. **Текущий выбор** подсвечен синим

**Пример использования:**
```
Введите: "план"
→ Фильтруется: F00002 Планы первичных продаж

Введите: "календ"
→ Фильтруется: D00001 Основной календарь

Клик на поле → Показывает весь список
```

## Технические детали

### Новый функционал:

1. **Кастомный комбобокс:**
   - State для comboboxValue и showDropdown
   - Фильтрация в реальном времени через useMemo
   - onFocus → очищает поле и показывает список
   - onBlur → скрывает список через setTimeout
   - onMouseDown → выбор элемента

2. **Направление связи:**
   - Определяется через `rel.fromTable === selectedEntityId`
   - Исходящие (→) - из текущей сущности
   - Входящие (←) - в текущую сущность

### CSS стили:

- `.combobox-wrapper` - относительное позиционирование
- `.dropdown-list` - абсолютный, z-index 1000
- `.dropdown-group` - группировка
- `.dropdown-item` - элементы списка с hover
- `.dropdown-item.selected` - текущий выбор (синий фон)
- `.direction-cell` - стрелка направления (синяя, жирная)
