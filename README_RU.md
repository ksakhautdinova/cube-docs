# 📊 Cube Documentation Tool v1.1.0

Система управления документацией OLAP куба данных.

---

## ⚡ Быстро

```bash
cd cube-docs
npm install
npm run dev
```

Откроется на http://localhost:3000

## 📊 Структура данных

### Факты (F-префикс)
Таблицы фактов с показателями:
- F00002 - Планы первичных продаж
- F00006 - Скидки по первичным продажам

**Содержат:** Меры с DAX формулами

### Измерения (D-префикс)
Справочники с атрибутами:
- D00001 - Основной календарь
- D00400 - Скидки
- D00750 - SKU

**Содержат:** Столбцы с типами данных

## ✨ Примеры использования

### Просмотр факта
1. Выберите F00002 в селекторе
2. Увидите показатели с DAX формулами
3. Увидите связи с измерениями

### Просмотр измерения
1. Выберите D00001 в селекторе
2. Увидите атрибуты (столбцы)
3. Увидите связи с фактами

### Создание требования
1. Перейдите на "Требования"
2. Заполните форму
3. Добавьте меры
4. Нажмите "⚡ Авто"
5. Экспортируйте DOCX
