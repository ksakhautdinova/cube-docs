```
   - PRE-FLIGHT (подготовка)

```

### Сценарий 3: Я code reviewer

```
1. Прочитай PLAN_SUMMARY.md полностью
2. Прочитай DETAILED_FILE_CHANGES.md полностью
3. Откройте PR в GitHub
4. Проверь что каждый файл соответствует плану:
   - Файл X создан/изменён согласно описанию
   - Код содержит правильные переменные (var(--primary-500) и т.д.)
   - Нет hardcoded цветов
   - Импорты правильные
5. Проверь что не было случайных изменений в других файлах
6. Approveй PR если всё OK
```

### Сценарий 4: Я архитектор, нужна история решений
